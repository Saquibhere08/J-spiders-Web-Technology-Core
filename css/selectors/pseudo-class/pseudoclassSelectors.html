Pseudoclass Selector

It is used to target the special state of an element

All the Pseudo class selectors are represented with :

    --------------------------------Dynamic Pseudo Class Selector-------------------------------------------

  • Link Pseudoclass Selector
  • Visited Pseudoclass Selector
  • active Pseudoclass Selector
  • focus Pseudoclass Selector
  • hover Pseudoclass Selector
  • checked Pseudoclass Selector

    • --------------------------------Structural Pseudo Class Selector-------------------------------------------

  • first-child Pseudoclass Selector
  • last-child Pseudoclass Selector
  • nth-child Pseudoclass Selector
  • first-of-type Pseudo-class Selector
  • last-of-type Pseudo-class Selector
  • nth-of-type Pseudo-class Selector
