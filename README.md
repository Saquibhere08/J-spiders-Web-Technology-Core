# Web Tech Basics 🚀

A beginner-friendly repository to learn the fundamentals of **HTML**, **CSS**, and **JavaScript** with simple examples and practice projects.

---

## 📚 Topics Covered

### 🌐 HTML
- Introduction to HTML
- Basic Tags
- Forms and Inputs
- Tables and Lists
- Semantic HTML
- Multimedia Elements

### 🎨 CSS
- CSS Syntax
- Colors and Backgrounds
- Box Model
- Flexbox & Grid
- Responsive Design
- Animations and Transitions

### ⚡ JavaScript
- Variables and Data Types
- Functions
- Loops and Conditions
- DOM Manipulation
- Events
- Basic Projects

---

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/web-tech-basics.git
```

2. Open the project folder:

```bash
cd web-tech-basics
```

3. Run HTML files directly in your browser.
